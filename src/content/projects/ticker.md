# Ticker - Sistema de Ponto Eletrônico Digital

## Visão Geral
Sistema de ponto eletrônico digital que modernizou completamente o processo de controle de frequência, eliminando registros manuais e proporcionando análises em tempo real.

## Tecnologias Utilizadas
- **Python** - Backend e automações
- **FastAPI** - API REST
- **PostgreSQL** - Banco de dados
- **Redis** - Cache e filas
- **Oracle Cloud** - Hospedagem

## Funcionalidades
- Registro de ponto via web e mobile
- Validação automática de inconsistências
- Relatórios gerenciais automatizados
- Integração com folha de pagamento
- Dashboard de análise de frequência

## Resultados
- Economia de 200+ horas/mês em processos manuais
- Precisão de 99.8% nos registros
- Redução de 95% em erros de lançamento
- Acesso em tempo real para gestores
