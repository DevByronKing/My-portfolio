# Sitec Web - Plataforma de Gestão Full Stack

## Visão Geral
Plataforma web completa desenvolvida para a SEDUC-PA que integra gestão de ponto eletrônico, análise de relatórios, controle de férias e múltiplos sistemas corporativos em uma interface unificada.

## Tecnologias Utilizadas
- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: Node.js, Express
- **Banco de Dados**: PostgreSQL
- **Infraestrutura**: Docker, Oracle Cloud
- **Autenticação**: JWT, OAuth 2.0

## Funcionalidades
- Dashboard centralizado com KPIs em tempo real
- Gestão de ponto eletrônico digital
- Automação de leitura de relatórios
- Sistema de controle de férias
- Integração com múltiplas APIs

## Impacto
- Atende 5.000+ usuários ativos
- Redução de 80% em processos manuais
- Disponibilidade de 99.9%
- Interface responsiva e acessível
